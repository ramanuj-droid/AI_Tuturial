Logistic Regressio

Classification → Predict categories
Logistic Regression is the foundation of classification in ML.
It solves classification problems.

| Input        | Output               |
| ------------ | -------------------- |
| Email        | Spam / Not Spam      |
| Transaction  | Fraud / Not Fraud    |
| Student data | Pass / Fail          |
| Medical data | Disease / No Disease |

We use Logistic Regression because-linear regression is not bounded.
