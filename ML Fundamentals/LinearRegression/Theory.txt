#Machine learning model knows it is wrong and how it measures that error,

| x | Actual y | Predicted y |
| - | -------- | ----------- |
| 1 | 8        | 5           |
| 2 | 11       | 8           |

Residual = Y(actual) - Y(predicted)

- MSE =  1/n sum of [(Residual)^2]

*Loss Function (Important Term)

The error metric used during training is called the Loss Function.
For Linear Regression: Loss = Mean Squared Error
Training goal: Find w and b that minimize the loss

Mathematically: min(1/n x sum(y -(wx + b))^2)

*GRADIENT DESCENT

How do we find the best values of w and b?

- That is solved by optimization, and the most important optimization algorithm in ML is Gradient Descent.

Gradient = direction of steepest increase.

But we want minimum loss, so we move in the opposite direction.

Update rule:

𝑤 = 𝑤 − 𝛼[∂𝐿/∂𝑤]
𝑏 = 𝑏 − 𝛼[∂𝐿/∂𝑏]

Where:
α      ==  learning rate 
∂L/∂w  ==  gradient 
L      ==  loss 

Learning Rate : Learning rate determines step size.

Too small : very slow learning
Too large : oscillation / divergence
good learning rate → smooth convergence

Typical values: 0.1 ,0.01 , 0.001

Loss function: 𝐿=1/𝑛 ∑(𝑦−(𝑤𝑥 +𝑏))

Derivatives become:

∂𝐿/∂𝑤 = −2/𝑛∑𝑥(𝑦−𝑦^))
∂𝐿/∂𝑏 = −2/𝑛∑(𝑦−𝑦^))

1 Initialize w and b randomly
2 Predict y
3 Compute loss
4 Compute gradients
5 Update parameters
6 Repeat until convergence
